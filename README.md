# ❣️ 프로젝트 소개

 **우리가 사랑하는 방법 🩷, MeloMeter**
 
### 🥲 수많은 커플 앱 이용에 지치셨나요?

> MeloMeter 하나로 연인과 위치를 공유하고, 기념일을 챙기고, 채팅과 백문백답도 즐겨보세요!
> 

### 🫵 전화번호만 있다면 누구나 이용할 수 있습니다.

> 전화번호 인증을 통해 발급된 초대코드로, 상대방과 연결할 수 있습니다.
>
